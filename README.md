# AuraSentinel-MVP

AuraSentinel is a privacy-friendly AI solution designed to detect early signs of tension, theft, or aggression in retail environments, using audio and video inputs.

🧠 Powered by a multi-modal detection system inspired by academic research (like CASSANDRA),  
🔁 Continuously improved through a simulated federated learning loop,  
📦 Built to be lightweight, responsive, and respectful of customer privacy.

## Features
- Audio-based emotion detection (stress, tension)
- Pose/movement analysis via skeleton keypoints
- Local feedback for improvement (simulated FL)
- Silent staff alerts, no facial ID
