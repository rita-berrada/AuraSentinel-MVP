# AuraSentinel-MVP

AuraSentinel is a privacy-friendly AI prototype designed to detect early signs of tension, theft, or aggression in retail environments, using simple audio and video inputs.

🧠 Built on a lightweight, multi-modal detection system,  
🔁 Continuously improved through a simulated federated learning loop,  
📦 Designed to be responsive, respectful of privacy, and easy to integrate into existing stores.

## Features (MVP scope)
- Audio-based stress/tension detection via spectrograms
- Pose/movement agitation scoring from video frames
- Local feedback loop to simulate model improvement
- Silent alert system for store employees (log-based)

🚧 MVP in development: the goal is to demonstrate core logic, detection flow, and privacy-respecting architecture.
